# tableau-training-effectiveness
An interactive Tableau dashboard project analyzing training effectiveness using participant feedback, assessment scores, and engagement metrics to evaluate trainer performance and improve learning outcomes.
# Training Effectiveness Dashboard (Tableau)

## Overview
This project analyzes the effectiveness of technical training programs using interactive Tableau dashboards. It evaluates trainer performance, learner engagement, and knowledge improvement using multiple performance indicators.

## Objectives
- Measure trainer effectiveness using feedback and assessment scores
- Analyze learner engagement and participation trends
- Identify high-performing and underperforming training programs
- Support data-driven decisions for training improvement

## Tools & Technologies
- Tableau
- Microsoft Excel / CSV datasets

## Key Metrics Analyzed
- Pre- and post-training assessment scores
- Participant feedback ratings
- Attendance and engagement levels
- Simulation and practical performance indicators

## Project Files
- Tableau Workbook: `Training Effectiveness Dashboard.twbx`
- Datasets: `data/` (anonymized / sample data)
- Screenshots: `screenshots/`

## Notes
Due to confidentiality, the data used in this project is anonymized and shared only for demonstration purposes.
